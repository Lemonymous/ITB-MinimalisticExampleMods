This mod showcases minimalistic examples for some ways to code custom ui in ITB

files:
	ui_examples.lua
		- simple frame
		- function chaining
		- ui tree-hierarchy
		- create ui on demand
		- ui variables
		- ui functions

- Menues can be seen in-game when the mod is enabled and the correct section has been uncommented
- Add a dash in front of --[[ to disable the comment block and enable the code inside
- Remove a dash from ---[[ when you want to disable the code and enable the next example
